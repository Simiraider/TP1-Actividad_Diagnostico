# TP1-FRONTEND

Autoria: Simón Flomenboim, 5 D.
Tematica: Catalogo de Viajes/Destinos.

Ítems resueltos
    
    Astro Framework: Estructura general y maquetación.

    Componentes: Uso de Navbar, Hero, Card y Row.
    
    Rutas Dinámicas: Páginas individuales por destino mediante slugs.
    
    Colecciones de Datos: Gestión de destinos en archivos JS.
    
    Estilos: CSS responsivo y estética de streaming.
    
    Repositorio: Público para entrega.
     

1-Primero Comence creando el proyecto y pense en hacer un temporizador pomodoro con flashcards y multiple choice, como una herramienta de estudio pero me parecio muy basico y decidi cambiar a un catalogo de viajes.

2-Luego continue con la implementacion de Astro, y la configuración de VS Code.

3-Comenze estructurando el html, js, css y los componentes de astro.

4- Pense como deberia ser la estructura del proyecto para que quede lo más estetico posible, eligiendo un catalogo al estilo netflix con un navbar, hero, cards y rows.

5- Al principio era para un solo destino, pero luego decidi hacerlo para varios destinos y agregarle rutas dinamicas para que cada destino tenga su propia pagina. Los destinos los pude haber puesto en una base de datos y agregar una funcion de usuarios pero me llevaba una semana minimo, y queria aprovechar el tiempo al maximo. Tengo cerca de 70 destinos todos puestos a mano. Antes se clickeaba y te llevaba a TripAdvisor, pero decidi hacer el cronograma hora por hora. Tambien en la pantalla de inicio puse Usuarios pero no guardan datos ninguno de ellos, son de muestra jeje.

6- También decidi agregar una seleccion de multiples paises y organizador perfecto para un viaje espectacular mediante Inteligencia Artificial, utlizando una API KEY de Gemini. Esta la tuve que borrar para que el repositorio lo pueda poner en publico.  

7- Por ultimo, decidi que se pueda exportar los viajes y ponerle el nombre personalizado, y que se pueda ver en una pagina de "Mis Viajes". Al exportar, se genera un archivo .ics que se puede importar en cualquier calendario. Outlook, Calendario de Samsung o de Google.

8- Decidi deployarlo en vercel al link: https://tp-1-actividad-diagnostico.vercel.app/.
