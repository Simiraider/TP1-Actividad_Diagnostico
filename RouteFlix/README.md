RouteFlix es una aplicación web pensada como un catálogo interactivo de rutas y destinos de viaje, inspirada en la forma en que plataformas como Netflix presentan su contenido. La idea es que explorar lugares se sienta tan simple y atractivo como navegar por una biblioteca de series o películas.

La interfaz utiliza un diseño en modo oscuro, con fondos profundos para que las imágenes de los destinos resalten visualmente. Las fotografías y videos de los lugares funcionan como el elemento central del sitio, acompañados por una tipografía moderna y limpia que mantiene la navegación clara y minimalista.

En la parte superior se encuentra una sección principal destacada, donde se muestra un destino con imagen o video de fondo y un botón para comenzar a explorar rutas relacionadas. Debajo aparecen distintas filas horizontales organizadas por categorías, por ejemplo rutas populares, escapadas cortas o destinos recomendados. Cada fila funciona como un carrusel de tarjetas que permite recorrer distintos lugares de forma rápida.

Cada destino se muestra en una tarjeta visual, que al pasar el cursor se agranda ligeramente y revela más información, como una breve descripción, clima o una estimación de costos del viaje. Este tipo de interacción hace que la experiencia sea más dinámica y cercana a una plataforma de streaming.

El catálogo se organiza a partir de datos almacenados en un archivo estructurado, donde se incluyen los nombres de los destinos, imágenes, categorías y descripciones. Esto permite generar las filas automáticamente y aplicar filtros o búsquedas en tiempo real para encontrar lugares según continente, tipo de viaje o presupuesto.

El proyecto se desarrolla con HTML y CSS para la estructura y el diseño, utilizando herramientas como Grid y Flexbox para organizar el catálogo visual, y JavaScript para manejar la carga de datos, las animaciones y la interacción del usuario. El resultado es una experiencia de exploración visual pensada para que descubrir nuevas rutas de viaje sea intuitivo y atractivo.

El nombre RouteFlix refleja esta idea de convertir la exploración de destinos en una experiencia similar a navegar una plataforma de contenido. Una alternativa posible es Roamly, que transmite una sensación más ligera y relacionada con la idea de viajar y moverse libremente.
